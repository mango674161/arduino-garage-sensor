# Arduino Automatic Cardboard Garage Door







A simple Arduino-based automated garage door that detects nearby objects using an ultrasonic sensor, plays sound effects with a passive buzzer, and opens a cardboard garage door using a servo motor while displaying the door's status with LEDs.







## Features



- Automatic object detection using an ultrasonic sensor

- Slow servo-controlled garage door movement

- Opening and closing sound effects using a passive buzzer

- Three-state LED status system
  - Red = Door Closed
  - Yellow = Door Moving
  - Green = Door Open

- Automatic door closing after a 2-second delay







## Technologies Used



- Arduino (C/C++)

- HC-SR04 Ultrasonic Sensor

- Servo Motor

- Passive Buzzer

- LEDs (Red, Yellow, Green)

- Resistors

- Breadboard & Jumper Wires
